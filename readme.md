## ✨ Features

- 🏠 **Dynamic Homepage** — Hero slider showcasing featured products (iPhone 14 Pro, MacBook Air, PS5), category browsing, new arrivals, popular products, and a seasonal sale banner
- 🛍️ **Products Listing Page** — Filter & browse products by category with grid layout, pricing, and quick-add functionality
- 📦 **Product Details Page** — High-quality images, color/variant selectors, quantity control, delivery info, detailed specifications, star ratings, customer reviews, and related products
- 🛒 **Shopping Cart** — Real-time cart management with item list, quantity updates, order summary, and checkout flow
- 📂 **Category Navigation** — Phones, Laptops, Cameras, Headphones, Computers, Gaming
- 💰 **Discounts Section** — "Discounts up to 10%" promotional area with product cards
- 🌙 **Dark & Light Theme** — Clean dark navbar with light content areas for a premium look
- 📱 **Responsive Design** — Fully optimized for desktop, tablet, and mobile
- ⭐ **Reviews & Ratings** — Aggregate star ratings with individual customer review cards
- 🔗 **Footer** — Multi-column footer with links, newsletter signup, and social icons

## 🚀 Getting Started

### Prerequisites

No build tools required — this is a static HTML/CSS/JS project.

### Installation

```bash
# Clone the repository
git clone https://github.com/Kuldip1211/Cyber.git

# Navigate into the project
cd Cyber

# Open in your browser
open index.html
# or simply drag index.html into your browser
```

### Live Server (Recommended)

If you use VS Code, install the **Live Server** extension and click **"Go Live"** for hot-reload during development.
